Floated Left

This paragraph wraps around the floated box. Floats allow text to flow alongside images or elements. It stays in normal flow while the floated element is removed from traditional positioning.

Float: right

Floated Right

When floated right, the text wraps on the left side. Floats are commonly used for image placement or old-school column layouts.

Cleared right — starts below the float.
